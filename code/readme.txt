
This software demos the algorithm in our paper published in ACM Symposium on Virtual Reality Software and
Technology (VRST 2019). Please check the codes and the paper for more details.

********************************************************************************************************************************
If you use this code and its associated data (e.g. images) in your publication, please cite the paper, Thanks:
Xianyong Fang, Jikui Yang, Jie Rao, Linbo Wang and Zhigang Deng. Single RGB-D Fitting: Total Human Modeling with 
an RGB-D Shot, ACM Symposium on Virtual Reality Software and Technology (VRST 2019), pp. 24:1-24:11, 2019
********************************************************************************************************************************

These codes optimizes the energy model and thus outputs the fitting result.

Input: Joint positions of the full body and hands, and the smooth human body point cloud.
Output: A huam model corresponding to the the subject in the shot.
